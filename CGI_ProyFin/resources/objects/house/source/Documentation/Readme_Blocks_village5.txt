Product Description:  
  
Blocks Village 5, it is my fifth home model for collection "Blocks Village", thrown in the beginning of the year of 2009.  
The model possesses textures in high resolution with 3000 x 3000 pixels and with 64761 polygons.   
  
For the package of models in the format Wavefront obj, they are separate from the following form:  
1) - Blocks_village5.obj  
2) - blocks_village5.mtl  
3) - blocks_village5.FBX

Usage Tips or Limitations:    
    
To use this model you need any program that can import *.obj files or FBX. Navigate to the obj files found in 'obj
file and textures'. You can try importing the"Blocks-village5.obj" or FBX. If your program solder the mesh that
like this you cannot manipulate separately, then carry the individual parts.   
  
If you import the separate parts, check your applications import options dialogue for settings that might
effect the final position. If settings like 'centre' or 'drop to floor' are enabled then the parts will not
assemble in the correct locations for you. you will have to move them into position after import.     
    
Try different bump settings to get different effects.  
  
With a bit of imagination you could use the parts to help make new buildings.
